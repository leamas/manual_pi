Toggle Tracking

Track Off / On

Toggling the Track icon on the Toolbar turns tracking on and off. Tracking means that a record is kept of the vessel’s position as it proceeds. When tracking is on, the button has a green background. A trail is left behind the vessel on the chart display. Tracking can also be enabled/disabled under the Navigation tab in the MENU BAR

Tracking is persistent. If you shut down OpenCPN with tracking active, tracking will be activated automatically next time you start. Note that tracking on start up is delayed until a GPS position is available. This action avoids large nonsense jumps at the beginning of the new track.

See Toolbar > Options > Own Ship > Tracks to configure options for Automatic Daily Tracks.

See Toolbar > Route and Mark Manager > Tracks to manage recorded tracks.

See Use Tracks to learn more about using Tracks.
