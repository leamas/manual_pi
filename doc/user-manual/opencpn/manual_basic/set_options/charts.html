Charts

Chart Overview

OpenCPN supports both Raster charts (Raster Navigation Chart or RNC) and Vector charts (Electronic Navigation Chart or ENC) in a number of different chart formats. However OpenCPN does not include charts, they must be downloaded separately. Depending on the chart coverage area and the responsible hydrographic authority, some charts are available for free, others must be purchased.

The different Raster chart formats supported by OpenCPN include BSB. KAP, and MBTiles. BSB & KAP are Raster chart formats originally developed by hydrographic authorities. You can find many sources for these raster charts. US NOAA is providing fewer Raster RNC Charts and is now using Vector ENC Charts.

MBTiles is a new storage mechanism for tiled map sets and supports both Raster and Vector charts. A benefit of MBTiles is that any tiled map format can be used, for example Google Earth images and Navionic’s Chart Images. Refer to the MBTiles section for further details.

The different Vector chart formats supported by OpenCPN include CM93, S57 and S63.

S57 is an international standard developed by the International Hydrographic Office (IHO) and is used to encode the data in electronic navigation charts. Examples of S57 charts include the free US National Oceanic and Atmospheric Administration (NOAA) chart sets.

S63 is an encrypted version of S57 designed to prevent unauthorised copying of charts and to ensure their integrity. Examples of S63 charts includes those from the UK Hydrographic Office (now known as Admiralty Maritime Data Solutions). To use S63 charts users require a S63 User Permit.

CM93 is an old chart format originally used by C-Map. Pirated copies of these charts are available on the Internet, however these charts are no longer maintained, are out of date, of dubious providence and therefore not recommended for safe navigation.

For charts freely provided by their respective hydrographic authorities, the Chart Downloader plugin simplifies the process by selecting chart areas, downloading and installing new and updated charts. The table below lists the different hydrographic authorities and chart sets that are freely available. Refer to the Chart Downloader section for details on how to use the plugin.

Charts for purchase by users of OpenCPN are available from o-charts. They have negotiated very attractive pricing and have extensive worldwide coverage. Charts available from o-charts include both S63 charts, OpenCPN Encrypted System Electronical Nautical Charts (oeSENC) and OpenCPN Encrypted Raster Nautical Charts (oeRNC).

oeSENC and oeRNC are charts that have been officially licensed by o-charts from their respective hydrographic authorities or chart distributors and then encrypted by o-charts for specific use by OpenCPN users. Similarly S-63 charts are sourced from the relevant hydrographic authorities and the official S63 User Permits obtained.

To view the chart coverage areas, to purchase and download charts visit the o-charts web site. Refer to the O-Charts section for details on how to use the plugin.

  • Chart Files OpenCPN does not come with charts, about Chart File system.

  • Vector Files S57 ENC, S63 Encrypted, CM93

  • Chart Groups Useful tool to group charts by path.

  • Chart Downloader Chart Downloader provides an easy way to download and install free NOAA and RNC and ENC charts around the world. It is the easiest way to get started with charts. Download a few charts in your area and use them to learn, then later add more as needed. NOTE: Free charts are not assured everywhere.
