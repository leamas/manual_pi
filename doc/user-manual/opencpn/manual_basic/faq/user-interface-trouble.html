User Interface Questions

Where are sound controls located now?

Not very obvious but sound controls moved in v5.6.0

  • Options→Ships→AIS: enable "Play sound on AIS……"

  • Then in Options→User interface→Sounds: Enable [x] "Play sound on AIS alert".

  • Select another sound file if desired.

  • The "Test" button is here as well.

Why is own ship pointing towards COG instead of directly towards the bow, and there is no magnetic heading?

Enable thw WMM World Magnetic Model please.

It makes no sense to use magnetic on true north facing mercator projections.

Ownship true north heading predictor is only displayed if true north value is available.

If HDT is not available in the nmea data stream we use HDM and variation from the WMM plugin to estimate an effective HDT.

Why is OpenCPN slow?

  • A disabled plugin, whether managed or un-managed, does not get loaded, and cannot be responsible for performance deterioration at run-time.

  • There is some very small cost involved in inspection of all available plugins, enabled or not, during OCPN startup. This is a transient, probably 50 msec. or so.

  • If you are having performance problems, look to the graphics system. Are you using OpenGL?

Jumping and Uncontrollable Mouse with Win10 and USB hub

See this CF Thread

  1. Solution 1: No need to edit registry. Works for all versions of Windows.

  2. Open device manager. (usually through control panel.)

  3. Expand "Ports (COM and LPT)"

  4. Right click on the port that is messing up the mouse and click properties.

  5. Click Port settings→Advanced and turn Serial Enumerator off.

  6. It should be ok to turn Serial Enumerator off on everything except the actual mouse.

  7. If the mouse not working, how can it be possible to do the above? Good question: Do this when the effect is intermittant.

Solution 2: Disable Serial Mouse Detection

How to enable Auto Follow

  • In the upper left corner to MenuBar, "Navigate" drop down, and check "Auto follow".

  • Use Ctrl-A or F2.

  • Lower right Boat should have a green circle when enabled.

  • Or click on the lower right Boat to turn on the green circle for AutoFollow.

Why doesn’t Auto Follow work?

  • If you pan the boat off-screen, auto-follow is automatically switched off.

  • Do you have the GPS connected and working?

  • Does the GPS indicator in the upper right show good green signals and operational GPS?

  • In the NMEA debug window (Options|Communication), do you observe a continuous data stream or does it stall?

  • Do you have regular repeating GLL or GGA messages?

  • Version 5 has a new feature panning the chart, the relative position of the boat "sticks". The boat snaps back to center when you turn auto-center OFF, which at first is confusing.

Is the closest we can Zoom into 1:800 ?

  • Without tweaks, Yes!

  • Tweak: In config file: [Settings] add/set: MaxZoomScale=800 or what you want "up" to 50. (1:50)

Where is the Zoom to Cursor setting?

  • Go to the Toolbox and enable "Zoom to Cursor" on the Display→General tab

How to increase the size, color and location for the Toolbar?

In Tools→User interface you can select to collapse the bar after a given time and further down adjust the size with the slider "User Interface scale factor". Color and location are though not adjustable. Dark grey on black is OK at night but in bright sun the icons are just not visible. For the same reason, you can change the status bar default black text on grey to white.

Why doesn’t the plugin work?

How are font sizes for vector chart Depth Soundings adjusted?

Out of the box, the "Depth" notation is unreadable.

  • Try Options > User Interface > Chart Object Scale Slider set on +4 or +5.

  • Try using the Swedish color palette which changes depth font to black.

  • The Chart Object Scale Slider is for Vector Charts only.

  • Raster Chart cartography is superior and depths are very clear in comparison.

How can all font sizes be easily changed to a better size?

  • Close Opencpn. * In the Opencpn.ini or opencpn.conf file find DefaultFontSize=0 under the [Settings] section, just below ShowMenuBar=0 * Change the defaultfontsize setting to the font size you would like use and save the file. * Load Opencpn and check your font sizes.

How are fonts adjusted for the AIS Target List?

  • The AIS Target List does not exist in UI > Options Fonts selection. AIS target list uses the "Dialog" font key in Options > User Interface, Fonts and Font Color..

Mouse rollover feature becomes disabled

'Enable Touchscreen interface' in Options/User Interface disables the rollover feature on my PC with a touchscreen. With this setting enabled, touching a target displays the target information. Without it enabled the rollover feature works.

"Rollover" is a mouse thing. One does not exactly "rollover" with a touchscreen. This could get very distracting to the overall UI. Thus, rollover is disabled for touch.

Why don’t AIS and Watchdog Sound Alarms work on RPI?

Distorted sound

  • The wav file MUST be 16 bit, mono 22050 Hz in order to play correctly. I’ve converted the sounds from mp3 to wav using:

mpg123 -2 -m -w out.wav in.mp3 This is on a Raspberry PI using OpenPlotter 1.1.0 alpha)

No Sound in RPI:

  • Disable the onboard audio in /boot/config.txt, edit files /usr/share/alsa/alsa.conf and /etc/asound.conf to point to card 1 (my USB speaker as shown in response to "aplay -l".

  • Change Tools > Options > User Interface > Sound Device Index to 1 and the result is Watchdog and AIS test alarms work fine.
