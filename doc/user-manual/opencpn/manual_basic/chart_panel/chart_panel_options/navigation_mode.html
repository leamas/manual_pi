Navigation Mode

The Chart Panel is always found in the lower right corner of a chart canvas. Click on the Hamburger.

Course icon indicators turned on using Ctrl-i toggle and are located in the upper left of the screen.

Chart Display Orientation and GPS Status Icon

North Up | Course Up | Heading Up

There are three main options for Chart Display Orientation as shown in gps_status.

  • North Up (blue color) indicates that the Chart Display is oriented with North at the top (planning).

    * It may also indicate //Chart Up// mode for skewed charts, if that option is selected.
* {{opencpn:manual_basic:chart_panel:chart_panel_options:course-up_0.png?nolink&30x28}} Course Up (red color) uses COG (Course Over Ground) to orient the chart to the course ahead,
    * so that the Own Ship icon points up on the display (for navigating with a GPS). For example, //Course Up// is useful when navigating a river or a canal.
    * //Course Up// rotation update period, can be set in [[opencpn:manual_basic:set_options:display#advanced|Options -> Display -> Advanced Tab]] which is useful to prevent the display jerking around in a chop.
* {{opencpn:manual_basic:chart_panel:chart_panel_options:chart-up-pink.png?nolink&28x28}} Chart Up Skewed raster charts are normally shown "//Chart Up//", as intended by design, but they can optionally changed to "Show skewed raster charts as North Up" in [[opencpn:manual_basic:set_options:display#advanced|Options -> Display -> Advanced Tab]].
    * If this option is **not**  ticked, the right-click toggle option "Course Up Mode/North Up Mode" will be replaced by "Course Up Mode /Chart Up Mode", for skewed charts.
    * If this option is selected, skewed charts are shown North-Up. In both cases, activating course up, works as expected, the difference is when course up is **not**  on.
* Heading Up is available in the [[https://opencpn-manuals.github.io/main/rotationctrl/index.html|RotationCtrl plugin]].

Change Chart Display Orientation

Toggle North Up / Course Up:

  • Click the North Up or Course Up icon

  • Right Click menu and then select a command.

  • Lower right Chartbar Hamburger -→ Navigation

  • The Course Up/North Up icons share a small dialog with the GPS indicator. These GPS and Course indicators can be toggled on/off using the "Ctrl + i" shortcut.

  • The red and the blue arrows always point North.

In the screenshot below the Course Up is active. COG is 41 Deg. Notice how all text on the vector chart, is aligned correctly. This only works in OpenGL mode. The red arrow, in the upper right corner, always points North.

If an electronic compass is available in the NMEA stream the own ship icon will be orientated according to this heading. The difference can be seen on the chart if there is a cross current, for example. The boat below is heading WNW but is being set SW. An AIS CPA, Closest Point of Approach is also visible.
