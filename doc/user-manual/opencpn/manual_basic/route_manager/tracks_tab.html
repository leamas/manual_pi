Tracks Tab

A track is a record of the vessel’s position as it proceeds. A trail is left behind the vessel on the chart display. The record includes distance, bearing, latitude, longitude, speed and a timestamp for each waypoint recorded. Recorded tracks are managed in the Tracks Tab found under Toolbar > [/opencpn/manual_basic/route_manager[Route Manager > Tracks Tab.
Toggle Tracking button on the Toolbar is used to start and stop tracking.
track_properties - View and handle individual tracks.
use_tracks - split, join and merge tracks. Copy text, reduce track data.
Options > Ships > Own Ship > Tracks - Configuration for capturing track data.

Filter

Enter text in the filter box to limit the number of visible entries. See Navigation Sharing > Export & Import GPX Files for several reasons why keeping a large number of routes and tracks loaded in OpenCPN is not a good idea.

Show

The visibility of tracks is controlled by clicking the "eye" on each line. Check the "Show All Tracks" box to make all tracks visible on the chart. Clear the checkbox to hide all tracks.

When a track is visible on the chart, right-click brings up a context menu. Select Properties from the context menu, and the Track Properties dialog box opens.

Right Click on a Track in the list

  • See Reduce Data in Use Tracks

  • See Copy Text in Use Tracks

Sort

Tracks can be sorted by clicking the column headers.

Tracks Tab Buttons

Start Track

The ./../tracking.html button on the Toolbar is used to start and stop tracking. Tracking can also be started by clicking "Start Track" on the Tracks tab in the Route and Marks Manager or by selecting Enable Tracking in ./../menubar.html > Navigation.

Properties

track_properties Opens the Track Properties Dialog Box for the selected Track. Can also be accessed by double-clicking the track.

Delete

Click on a track to select it. Multiple tracks can be selected using CTRL+Click (CMD-Click on MacOS). Once selected, click the Delete button.

Import GPX

See Navigation Sharing

Export Selected

Select the desired track to export it to a GPX file. Selecting multiple tracks will merge them into a single GPX file. More information on import and export is found in See Navigation Sharing > Export & Import GPX Files.

Route from Track

Creates a route from track data. The new route will appear in the Route & Mark Manager and will have the same name as the track. Depending on the precision used to create the track, it is recommended to use the Reduce Data function on a track prior to converting it to a Route. This will reduce the number of waypoints.

Send to Peer

See Navigation Sharing

Delete All

Deletes all tracks.
