Track Properties

Basic

Name

  • The name of a recorded track defaults to the date it was recorded. The name can be modified here.

From/To

  • Enter a starting point and an end point for the track.

Display Parameters

Show On chart

  • Check this box to have the track appear on the navigation chart.

Color

  • Use the default color (set in Options>Ships>Own Ship)

Style

  • Change the default to display a different line style.

Width

  • Set the width in pixels to display a wider highlight.

Statistics

  • Reports the total distance, average speed and time enroute. Not editable.

Recorded Points

Shows a list of the recorded points. The table shows Leg, Distance, Bearing, Latitude, Longitude, Timestamp, and speed.

Print

Print the list of recorded points. A list of elements to print appears: Position, Course, Distance, Time, Speed.

Split

To split a track, select the track in the Track List. Click Properties. Select the desired waypoint from the Recorded Points table. Click Split. The tracks are split, with each segment name appended with _A and _B.

Extend Track

Creates a new, combined track by connecting the first time-stamped waypoint in the selected track with the last time-stamped waypoint in the most recent visible track in the track list. The Extend Track function deletes the first track from the Track List and appends _plus to the second, combined track.

For more detail, see Use Track > Extend Track.

To Route

Convert a track into a route. This is the same function as Route Manager>Tracks Tab>Route from Track.

Export

Export as gpx file.

Copy all as Text

Copy to the clipboard.

Advanced

Description

Add descriptive text or create a link.

Links

Add internet links such as links to a marina website or a web page about a point of interest.
