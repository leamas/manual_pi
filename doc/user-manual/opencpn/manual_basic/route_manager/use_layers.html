Use Layers

This section describes the use of layers. Layers are used to load a set of marks, waypoints, tracks, or routes from a single gpx (Global Positioning System Exchange) format file. All elements of a single layer can be displayed or hidden with a single button click. They are immutable (not editable), preventing accidental modification.

Note that marks and waypoints in a gpx file can be imported as a layer or as a set of individual waypoints. OpenCPN can get slow if it has to maintain information about many waypoints. A layer is treated as a single object, which is more efficient for OpenCPN.

The Layers user interface is covered in the Layers Tab and Layers Properties. More information about the use of layers is in Navigation Data Backup and in Use GPX Files.

Layer Characteristics

  1. Layers are not editable.

  2. A layer can be persistent and usable over multiple OpenCPN sessions.

  3. A layer can be temporary and only exist for the current session.

  4. Layers may be imported or exported.

  5. A waypoint in a gpx file for example, can be imported either as a “waypoint” or as an element in a layer. The difference is that when imported as a layer it cannot be changed or deleted.

Layer Uses

  • Layers can be used to update charts with new navigation marks and dangers as per Notices to Mariners, and to add Radio Call-In Points or to build a database of lights and locations, that will be visible on chart regardless of current scale.

  • The picture above displays a very good celestial fix, plotted with a gpx script saved as a layer.

  • It’s also possible to share (other) POI’s (Point Of Interest) by use of a layer. These could for example be mooring spots, places to bunker water or fuel etc.

  • More advanced uses could, for example, be an interactive cruising guide using the possibility to embed links in waypoint descriptions.

When is a feature a part of a Layer?

  • Layers are immutable, permanent, nothing can be changed. This could make them confusing, unless you are aware of the fact that they are layers!

  • To see if a waypoint, route or track, is a part of a layer, right-click the object. If it is part of a layer, the context sensitive menu’s title will start with the word "Layer" (see example "Layer Route" below). Alternatively, open properties and check the top of the dialog. If part of a layer, the layer’s name is printed. As always, double clicking on an object directly brings up the properties dialog,

  • A route belonging to a layer is also identified as such, by just hovering with the cursor above the route.

  • A Route belonging to a layer also has the option "Activate". It can thus be used, but still, no changes can be made. If you need the ability to edit a route, go to the Routes Tab and import the gpx file as a route.
