Marks Properties

All marks have properties that can be seen by right-clicking and pressing "Properties", or even simpler just double click the mark. Alternatively they are available in the Marks Tab of the Route Manager.

  • Basic

  • Links

  • Description

  • Extended

  • Defaults

Basic Tab

  • Name: Give the mark a suitable name. This will be visible on the chart, but can be hidden by un-ticking the Show Name box.

  • Icon: Change the default icon for the mark. Only the "Name" will be displayed if selecting the "Empty" icon.

  • The position of the mark: You can drag the mark with the mouse to the desired position or enter the correct Latitude and Longitude in these fields.

  • Right clicking anywhere in the latitude or longitude display gives you the sub dialog shown above. Latitude and longitude can be copied separately or together. A copied lat/long will be available in the paste buffer, and will be displayed if you activate the "Jump to position…​" dialog. It is also possible to paste a position. A Paste will try to be smart about parsing the format, and accepts a lot of different syntaxes. Even the GPX <wpt lat="0.00" long="0.00"> should work.

  • Description: Enter anything about the mark. Long descriptions can be read easier by switching to the Description Tab. See examples below.

  • Links: Marks can, as an option, have embedded links to the Internet or the the local computer. See below.

Links

Links (to websites or local files) are either opened in the default browser or through the default application, for example for pictures or pdf files. For security reasons, the links do not work for launching other programs.

Create a link by right clicking in the link area and Add New. Enter the link and a good description in the new dialog.

Enter an Internet address in URL or find a local file through the button Local File..

Edit links by by right clicking in the link area and pressing the "Edit" button.

Description Tab

This is simply an expanded view of the Description box in the Basic tab.

Extended Tab

Show on Chart: Untick this box, and the mark will not show on the chart. To make it visible again, use the Route Managers Mark Tab. Click the "Icon" column, to the far left, on the line with the Mark.
GUID is a unique identifier for the mark.

Show Mark name name will be visible in chart
Range rings rings will drawn around mark at given distance
GUID unique identifer of navigational object
Tide Station related tide station, if applicable

Defaults

Press the preferences wheel in the lower right to set preferences for future marks. A dialog box "save some defaults" will pop up.
