O-Charts.org

If there are no free charts for your area through Chartdownloader, check:

  • O-Charts.org

The most reasonable and easiest way to purchase charts for most worldwide cruising grounds. O-Charts is a service provider for OpenCPN charts, and is one very good reason to use OpenCPN.

  • FAQ Frequently Asked Questions

  • Conditions.

  • From the service provider: "Charts expire after a year from the purchase date. After expiration, you cannot request updates, and it is not possible to assign an expired chart to any other device, even if it still has free assignments."

  • O-Charts downloaded to OpenCPN remain available for use on the device after expiry.

Note: OpenCPN tries to avoid suppliers with subscription models which make charts unusable at the end of the license period.

Install the O-Charts Plugin - Start by installing the O-Charts Plugin that enables OpenCPN to read encrypted raster and vector charts.

  • Create account on O-Charts.org.

  • Purchase charts.

  • Follow the instructions.

  • Hint: The license gets linked to a specific device during installation. An user-chosen "System Name" is entered on O-Charts' online 'shop'. Chose the actual device name and identifier that will host the license to ease recall of which device the license has been installed on. Names like "SV-WAVY-Win10-Asus-NavComp" are more helpful than "SV-WAVY" (15-characters maximum).

Exception: Android automatically generates the name… Be sure to make note of it.

Install the O-Charts Plugin

This plugin enables OpenCPN to use encrypted charts purchased from O-Charts.org, a service provider for OpenCPN charts where free ones are not available.

After getting the plugin you can purchase charts at O-Charts.org:

  • Create account on O-Charts.org.

  • Purchase charts.

  • Follow the instructions.

  • Please note:

    • The license gets linked to a specific device during installation.

    • An user-chosen "System Name" is entered on O-charts' online 'shop'.

    • Choose the actual device name and identifier that will host the license to ease recall of which device the license has been installed on. Names like "SV-WAVY-WinAsus" are more helpful than "SV-WAVY" (15-characters maximum).

    • Exception: Android automatically generates the name… Be sure to make note of it.
