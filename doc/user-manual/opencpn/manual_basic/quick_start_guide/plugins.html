Plugins

Plugins are utilities that provide additional functionality to the core features of OpenCPN. Plugins are easily installed using the Plugin Manager (PIM). In addition to extending the functionality of OpenCPN, plugins integrate with OpenCPN by adding overlays to the Chart Canvas, adding new icons to the hamburger menu, new context menu items and new dialogs.

There are four core plugins that are always included with OpenCPN:

  • ChartDownloader

  • Grib Weather

  • World Magnetic Map

  • Dashboard

There are many additional plugins, developed and maintained by volunteers. Because these additional plugins are developed independently, they may not always be updated at the same time that new versions of OpenCPN are released and there may be some delay before they are made available.

The Plugin Manager is invoked from the Options dialog.

Periodically you should updated the Plugin Catalog to keep the list of available plugins up to date.

For more information regarding these additional plugins refer to the following links:

  • List of Plugins (Basic User Manual)

  • How to Install and Enable Plugins, (Basic User Manual)
