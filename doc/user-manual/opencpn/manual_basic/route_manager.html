Manage Routes and Marks

Route & Mark Manager lists routes, tracks, waypoints, and layers that are currently loaded into OpenCPN. It allows you to view and modify those navigation objects and their components.

Getting Around

Persistence

Navigation objects in OpenCPN, except for "Temporary layers", are persistent between OpenCPN sessions. Navobj.xml Data Backup discusses data persistence more fully, and how to backup this data.

Eye Show/Hide

Toggle visibility of active objects by clicking the "eye" to the left of the object name on each line. For waypoints, the eye is replaced by the waypoint icon shown on the chart. A red X across the "eye" (or waypoint icon) indicates that the object is invisible.

Column Sort

Sort objects by their column header. For example, clicking "Route Name" causes the routes below to be sorted alphabetically. Clicking again reverses the order. Also sort with To or From

Select with Ctrl-Left Click

Select multiple objects in the Routes, Tracks, or Waypoints tabs by holding down the CTRL key and left-clicking the objects or by holding the SHIFT key and selecting the first and last object of a group.

Properties by Double Click

Double-click on a route, track, or waypoint opens the Properties dialog (a shortcut for selecting the object and clicking on the Properties button).

Export & Import GPX Files

Export and import routes, tracks, waypoints, and layers using the GPX transfer format. Save your routes and marks to external files creating backup library - for example, to do route planning or archive existing data. Refer to Navigation Sharing - Export and Import of GPX Files
